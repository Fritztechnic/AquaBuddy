# 💧 AquaBuddy – Dein lustiger Wasser-Trink-Buddy! 💧

Eine spaßige JavaFX-App, die dich regelmäßig daran erinnert, Wasser zu trinken – mit lustigen Sprüchen, coolen Animationen und einem modernen Ocean-Dark-Design!

## ✨ Features

- 🥤 **Wasser-Tracking** – Zähle deine Gläser und verfolge deinen Fortschritt
- ⏰ **Timer-Erinnerungen** – Konfigurierbare Erinnerungen (5-120 Minuten)
- 😂 **Lustige Sprüche** – Über 50 witzige Nachrichten rund ums Wassertrinken
- 🎉 **Animationen** – Partikel-Effekte, Konfetti und Wassertropfen-Animationen
- 🔥 **Streak-System** – Verfolge deine Trink-Streak über mehrere Tage
- 💡 **Wasser-Fakten** – Interessante Fakten über Wasser und Hydration
- 🌊 **Ocean Dark Theme** – Wunderschönes dunkles Design mit Blau-Tönen
- 🎯 **Einstellbares Tagesziel** – 4 bis 15 Gläser pro Tag

## 🚀 Starten

```bash
# Im Projektverzeichnis:
./gradlew run
```

## 🛠️ Tech Stack

- **Java** mit **JavaFX 20**
- **FXML** für das UI-Layout
- **CSS** für das moderne Styling
- **Gradle** als Build-Tool
- **AnimateFX** für zusätzliche Animationen

## 📸 Wie sieht's aus?

Die App hat ein **Ocean Dark Theme** mit:
- Tiefsee-Dunkelblau als Hintergrund
- Leuchtende Cyan/Blau-Akzente
- Animierter Wassertropfen
- Glühende Progress Bar
- Konfetti bei Ziel-Erreichung! 🎊
